-------------------------------- IFRAME ---------------------------------------
	Quadro em linha que serve para abrir outro site dentro do seu site.	

	Tamanho padrão = 300x150 px.

	<iframe src="" frameborder="0" width="500" height="500">
		<a href="" target=""></a>	# Caso o dispositivo não suporte iframe isso será mostrado, caso contrário ficará oculto.
	</iframe>

	----- borda:
		<iframe src="" frameborder="
					    0	# Sem borda.
					    1	# Com borda (qulaquer valor != de 0).
		">

	----- scroll:
		<iframe src="" frameborder="0" scolling="
							auto	# Caso precise irá criar.
							yes	# Irá criar mesmo não precisando.
							no	# Não irá criar mesmo precisando (bugado em alguns navegadores).
							">

	----- Mudando o iframe:

		<ul>
			<li><a href="001.html" target="frame"></li>
			<li><a href="002.html" target="frame"></li>
			<li><a href="003.html" target="frame"></li>
			<li><a href="004.html" target="frame"></li>
		</ul>
		<iframe id="tela" name="frame" srcdoc="<h1> Escolha uma das opções acima </h1>">	# srcdoc aceita html e será a mensagem mostrada antes do usuário escolher um site.
		</iframe>
